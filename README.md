# awesome-born2blegend
Clash of awesomeness

Install angular cli
-npm install -g @angular/cli
-npm install

Install firebase 
-npm install -g firebase-tools
-firebase login

Steps to deploy to production:
-ng build
-firebase deploy
    or firebase deploy --only hosting
